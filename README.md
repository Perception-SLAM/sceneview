Sceneview

Sceneview is a visualization toolkit for real time rendering in scientific,
engineering, and robotics applications.

## Supported platforms
- Ubuntu (14.04+)
- OS/X (via Homebrew)

## Requirements
- CMake
- Qt 5.4+
- Assimp
