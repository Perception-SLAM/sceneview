Detailed Description

Pure virtual class that all scene graph nodes inherit.

The SceneNode class maintains state common to all scene graph nodes:

  • Node transform.
    • Represented as Translation * Rotation * Scale.
  • Node name.
  • Node visibility.
  • Parent node.
